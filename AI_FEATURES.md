## 5. 수입·은퇴 예측 엔진 (1순위) — 기획서 AI 목록의 첫 항목

**왜**: 은퇴 곡선(2041~2044)이 지금 프론트 하드코딩. "입금 추세 + 커리어 추세로 우리만의
예측"이 서비스 차별화의 핵심 주장인데 실물이 없다.

- **입력**: 원장의 income 시계열(월 합계·입금 간격) + (데모) 커리어 활동 추세 상수
- **출력 ①** `다음 수입까지 예상 공백`: 입금 간격 분포(중앙값·백분위)로 다음 입금 예상일 밴드
  → 가계부 "다음 수입까지 N일" / 코치 컨텍스트
- **출력 ②** `은퇴 곡선 밴드`: 소득 추세 외삽 × 활동 추세 가중(활동=선행지표) →
  목표 노후 생활비와의 교차 시점을 **점이 아닌 신뢰구간**(예: 2041~2044)으로
  → 홈 은퇴 카드·Retirement 화면 라이브 수치화
- **AI vs 결정론**: 계산은 전부 시계열 통계(지수평활·외삽 — 공모전 기준 ML 인정),
  해석 문구는 코치 LLM. 숫자 생성에 LLM 금지.
- **API**: `GET /v1/forecast` (공백·은퇴밴드·가정 노출)
- **가드레일**: 신뢰구간으로만 제시(단정 금지) · 하락 예측을 여신 거절 근거로 쓰지 않음
  (§15 정보적 위해 감시항목) · assumptions 전부 노출
- **파일**: `services/forecast.py` + `routes/forecast.py` + Home/Retirement 연결
- **공수**: 중 (반나절)

## 6. 포트폴리오 O/X 진위검증 (2순위) — v5 확정 사항의 실물

**왜**: Connect 화면에 "AI 진위검증 완료 ✓ +20점" UI만 있고 API가 없다.
"AI가 O/X만 보고 원본은 보관하지 않는다"는 v5 확정 문구를 시연 가능하게.

- **입력**: 파일 메타(이름·크기·해시·작성/수정 시각) + 설명 텍스트 (원본 파일은 저장 안 함)
- **판정 캐스케이드**:
  1. 결정론 체크 — 빈 파일·중복 해시(재탕)·메타 모순(수정시각 < 생성시각)
  2. LLM 판정 에이전트(2.4B) — 설명 텍스트가 백지·무성의·복붙인지 **O/X만** (점수 금지)
- **출력**: `{verdict: O|X, signals[]}` — X여도 감점 없음(only-upside), O만 +20점
- **가드레일**: 원본 미보관(해시만) · O/X 이진(정성 점수 금지 — 성실도 2층 구조와 일관) ·
  LLM 다운 시 결정론 체크만으로 O 판정 보류
- **API**: `POST /v1/portfolio/verify` + Connect '포트폴리오 직접 올리기' 연결
- **공수**: 중 (2~3시간)

## 7. 개인화 ①② 완성 (3순위) — "개인화 3종" 완결

**왜**: 강점 한 줄(③)만 완성. ①②가 있어야 "개인화 3종"을 완결형으로 발표.

- **① next-best-data**: 연결 상태(store.conn) → 미연결 소스 중 최대 가점을 이유와 함께 추천
  → 홈 '다음 할 일' 카드 동적화 ("홈택스 연결하고 +30점" → 연결하면 다음 추천으로 바뀜)
- **② 상품 CTA 개인화**: 상황(나이·소득구간·검증 단계) → 하나 상품 1개 + 한 줄 이유
  ("만 34세↓·저소득 → 하나원큐 햇살론유스: 3.3% 신고로 요건 충족")
  → Products 히어로 동적화
- **AI vs 결정론**: 둘 다 스코어링 룰(설명가능성) — 이유 문구만 코치 톤 템플릿.
  ⚠️ 심사점수 개인화 아님(경험·추천까지만 — §6-1 원칙)
- **위치**: 프론트 store 파생값 + (선택) `GET /v1/personalize`
- **공수**: 소 (1~2시간)

## 8. 챗 인텐트 라우터 (여유 시)

- 사용자 메시지 → 2.4B가 인텐트 분류(배분/세금/투자/일반) → 맞는 컨텍스트만 주입
- 효과: 코치 답변 정확도↑ + "라우팅도 에이전트" 오케스트레이션 스토리
- 공수: 소. 단, 데모 체감이 작아 5·6·7 완료 후에만.

## 구현 순서와 이유

```
5 예측 엔진  →  6 포트폴리오 O/X  →  7 개인화 ①②  →  (8 라우터)
```

- **5부터**: 기획서 AI 목록 1번 항목의 실물이자 은퇴 축(꺼내쓴다)의 데모 근거 — 공백이 가장 큼
- **6 다음**: v5에서 멘토링으로 확정된 문구의 시연 — Q&A 방어 소재
- **7은 짧게**: 룰이라 빠르고, 마무리로 "3종 완결" 선언용
- 각 기능은 독립 PR (feature 브랜치 → CI 초록 → develop), 골든셋/테스트 동반

## 공통 제약 (전 기능)

- 외부 AI API 금지 — Ollama 로컬(EXAONE 2.4B/7.8B)만
- 돈·점수 숫자는 결정론이 계산, LLM은 판단(O/X·선택·분류)과 말만
- 모든 판정에 signals, 모든 계산에 assumptions 노출
- LLM 다운 시 결정론 폴백 — 데모 불사
